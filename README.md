# Фінальний проект з дисципли "Реляційні бази даних"

## 18.11.25 - 28.12.25

### Аналіз даних інфекційних захворювань (SQL)

Цей проєкт реалізує базу даних `pandemic` для аналізу статистики захворюваності.
Виконано імпорт та нормалізацію "сирих" даних до 3НФ (таблиці `countries` та `infectious_cases_norm`).
Розроблено SQL-скрипти для статистичного аналізу, розрахунку часових інтервалів та власну функцію обчислення різниці років.
